template<typename T >
my_enable_if_t< my_is_integral_v< T > &&!my_is_string_v< T >, std::string > to_string_ip ( T  ip)

Prints a conditional IP address from an integral value.

This overload is enabled only for integral types using a custom SFINAE mechanism. The value is interpreted as a sequence of bytes and printed in unsigned form, starting from the most significant byte.

Each byte is separated by a dot (.), similarly to a traditional IP address representation. The number of printed elements depends on the size of the input type.
